# MusicDownloader 用户使用说明

## 一、使用前请先了解

本程序是一个辅助操作工具，用来帮助你把原本需要手动点击完成的下载流程进行自动化处理。

请仅在以下前提下使用：

- 你对目标资源拥有合法访问权限
- 你有权下载对应歌曲或文件
- 你理解并接受目标网站、夸克网盘等平台自身的使用规则

如果目标站点页面结构发生变化、下载入口变化、账号权限变化，程序可能无法正常下载。

## 二、首次使用建议

首次使用时，建议按以下方式操作：

1. 先打开程序
2. 检查下载目录、浏览器目录、站点配置是否正确
3. 先不要批量下载太多歌曲
4. 先用 1 首歌曲做测试
5. 确认流程正常后，再进行批量下载

如果程序首次运行后没有成功下载，优先检查：

- 是否已经登录夸克网盘
- 当前夸克账号是否可正常访问对应资源
- 当前站点配置参数是否匹配目标页面
- 目标页面是否仍然保留原有下载入口

## 三、为什么需要登录夸克网盘

程序下载过程中，最终文件通常不是直接从搜索站点本身下载，而是会跳转到夸克网盘或相关分享页，再从夸克网盘页面完成下载。

因此，登录夸克网盘的原因主要有这些：

- 某些资源必须登录后才能看到完整下载入口
- 某些分享页未登录时会限制下载
- 某些页面会要求登录后才能继续跳转或触发浏览器下载
- 某些账号状态异常、风控、封禁时，页面会拒绝下载

也就是说，程序本身不能绕过夸克网盘的登录要求。

如果夸克页面本来就要求你手动登录，那么程序运行时也同样需要你先登录，之后程序只是复用这个登录状态继续执行后续点击流程。

## 四、程序实际做了什么

这个程序只做了下载流程自动化，没有做额外操作。

当前自动化范围只有：

1. 根据你输入的歌名进行搜索
2. 自动进入对应搜索结果页面
3. 自动跳转到网盘相关页面
4. 自动进入歌曲详情或文件列表页面
5. 自动触发下载按钮，完成下载流程

你可以把它理解成：

它只是把原来需要你手动一下一下点击的流程，改成了程序帮你按顺序执行。

## 五、程序没有做什么

为了避免误解，这里特别说明：

本程序没有做以下任何事情：

- 没有破解网站
- 没有绕过登录验证
- 没有绕过夸克网盘权限控制
- 没有篡改页面内容
- 没有扫描你电脑里的其他文件
- 没有上传你的本地文件
- 没有批量抓取与你当前操作无关的数据
- 没有执行下载流程之外的其他自动化行为

程序只是在你给出歌名后，把“歌名搜索 -> 页面跳转 -> 网盘跳转 -> 歌曲详情 -> 下载”这一段流程自动化了。

除此之外，没有做其他任何操作。

## 六、登录状态说明

程序会复用本机浏览器的登录状态或程序自己的浏览器配置目录。

这意味着：

- 你首次登录后，后续一般不需要每次重新登录
- 如果你主动退出夸克账号，程序下次运行时可能需要重新登录
- 如果登录失效、账号异常、站点风控，程序也可能无法继续下载

## 七、下载失败时常见原因

如果下载失败，通常优先排查这些问题：

- 夸克网盘未登录
- 夸克账号状态异常
- 分享页已失效
- 站点页面结构发生变化
- 默认站点配置参数不再适配当前页面
- 下载按钮需要新的确认方式
- 目标资源本身已经被删除或限制访问

## 八、建议给最终用户的提醒

建议你在分发打包版程序时，提醒用户：

- 首次使用前先登录夸克网盘
- 先用单首歌曲测试
- 不要在不具备下载权限的场景下使用
- 如果页面变化导致下载失败，需要更新程序配置

## 九、结论

请把这个程序理解为一个“流程自动点击工具”。

它做的事情非常有限，只是把以下流程自动化：

歌名搜索 -> 页面跳转 -> 网盘跳转 -> 歌曲详情 -> 下载

除了这段流程自动化之外，没有做其他任何操作。
